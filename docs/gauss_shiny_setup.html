Here we give instructions for starting the Shiny server which will create an interactive plot from the results of the Gaussian mean estimation simulation study.

First, you need make sure that the following R packages are installed on your computer: dplyr, ggplot2, cowplot, shiny and rmarkdown. See the “Session Info” at the bottom for the versions of the R packages we used. Once you have installed these packages, run this code to verify that the packages are installed successfully:

library(dplyr)
library(ggplot2)
library(cowplot)
library(shiny)
library(rmarkdown)

Now you are ready to create the Shiny server with this command:

rmarkdown::run("../shiny/gauss_shiny.Rmd")

The interactive plot will be launched in your Web browser, or in an RStudio panel, depending on whether you are using R or RStudio.
